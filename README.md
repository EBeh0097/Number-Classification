# Hand written numbers classification



<h1>Data Mining - Optical Recognition of Handwritten Digits </h1>


<h2>Description</h2>
Optical Recognition of Handwritten Digits. The dataset includes 1,797 samples of 8x8 pixel images, suitable for classification tasks. The predictors are a NumPy array of shape (1797, 64) containing the flattened 8x8 pixel values for each image. Target is a NumPy array containing the ground truth labels (0-9) for each image. Classification is done using a Multi-Layer Perceptron (MLP).

<br />
<h2>Data Source</h2>
